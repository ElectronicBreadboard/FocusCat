# `@repo/ui`

Shared UI component library with Tailwind CSS theme and components.

## Installation

### For Projects Using Tailwind CSS

Import the theme (design tokens + global styles, no Tailwind bundle):

```css
@import 'tailwindcss';
@import '@repo/ui/theme.css';

@source '../node_modules/@repo/ui';
```

### For Projects NOT Using Tailwind CSS

Import the full Tailwind bundle with theme:

```css
@import '@repo/ui/tailwind.css';
```
